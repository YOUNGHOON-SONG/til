2. 1주차 주요 개념 키워드 적어보기
1주차 주요 키워드
버전관리 : 프로젝트 상태가 변경되는 정보를 알고 있다는 것.
git initialize : git이 관리하는 프로젝트 만드는 것
commit : 현재 프로젝트상태를 찰칵 저장하는 것 (누가, 언제, 변경내용)
adding/staging : commit에 반영할 변경파일 선택
local repository : 개인 컴퓨터 등 저장된 프로젝트
remote repository : 원격 저장소 (github) 에 저장된 프로젝트
tracking : 로컬 repo와 원격 repo있을 경우 연결하는 것
pull : 로컬repo -> 원격repo 반영
push : 원격repo -> 로컬repo 반영
clone : 원격repo만 있을 경우 로컬repo로 받는 것
